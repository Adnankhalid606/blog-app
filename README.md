# Blog App

A backend blog application built with Node.js, Express.js and MySQL.

## Roles

- User
- Author
- Admin

## Current Features

- Blog CRUD API
- Authentication
- Authorization
- Protected Routes
- Role-Based Access Control (RBAC)
- Author Request Workflow
- Admin Approval System
- Environment Variables
- Global Error Handling
